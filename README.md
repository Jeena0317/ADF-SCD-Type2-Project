# Azure Data Factory - SCD Type 2 Implementation

## Project Overview
Implemented Slowly Changing Dimension (Type 2) logic using Azure Data Factory Data Flow.

## Features
- Hash-based change detection
- Exists transformation
- Alter Row transformation
- Active/Inactive record management
- Historical tracking

## Tools Used
- Azure Data Factory
- Data Flow
- GitHub Integration

## Scenario
Tracks subscription changes for users and maintains full historical records.
